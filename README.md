# Jogo do número secreto

Um jogo simples de adivinhação de número secreto feito em JavaScript, HTML e CSS.

## Como funciona

O usuário deve tentar adivinhar o número secreto gerado aleatoriamente entre 1 e 30. A cada tentativa, o jogo informa se o número secreto é maior ou menor que o chute. O jogo conta o número de tentativas e permite reiniciar para jogar novamente.

## Recursos

- Interface responsiva e estilizada
- Feedback por voz usando [ResponsiveVoice](https://responsivevoice.org/)
- Imagens e efeitos visuais
- Controle de tentativas e reinício de jogo

## Como rodar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Divirta-se tentando adivinhar o número secreto!
